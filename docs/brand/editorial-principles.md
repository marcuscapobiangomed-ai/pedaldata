# Princípios Editoriais — Pedal Data

## Independência editorial
Nenhum parceiro comercial tem influência sobre:
- Conteúdo publicado
- Dados apresentados
- Resultados de comparações
- Posicionamento em rankings
- Recomendações

## Transparência de método
Todo artigo informa seu método de produção:
- Análise documental (pesquisa em fontes oficiais)
- Teste próprio (uso por mais de 30 dias)
- Análise mista
- Conteúdo com assistência de IA (sempre revisado)

## Uso de dados
- Toda afirmação técnica tem fonte registrada
- Dados de terceiros são citados com referência
- Dados proprietários são identificados
- Comparações usam mesma fonte e mesmo período

## Correções
- Erros são corrigidos com nota pública
- Correções não são feitas em silêncio
- Artigos corrigidos mantêm histórico da alteração
- Toda correção tem data e responsável

## Conflitos de interesse
- Links de afiliados são identificados
- Conteúdo patrocinado é rotulado
- Parceiros não pagam por menção positiva
- Amostras de produtos não garantem cobertura

## Relacionamento com marcas
- Marcas não aprovam conteúdo antes da publicação
- Marcas não pagam para remover dados
- Marcas podem fornecer informações técnicas, mas não controlam a análise
- Parcerias comerciais são separadas da produção editorial
