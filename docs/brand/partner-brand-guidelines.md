# Diretrizes de Marca para Parceiros — Pedal Data

## Uso da marca por parceiros
Parceiros podem mencionar o Pedal Data em:
- Site próprio (com link)
- Redes sociais (com @menção)
- Materiais de venda (com aprovação prévia)
- E-mail marketing (sem sugerir endosso)

## O que não é permitido
- Sugerir que o Pedal Data endossa o parceiro
- Usar o logo sem aprovação
- Alterar dados ou resultados do Pedal Data
- Usar a marca para descreditar concorrentes
- Publicar conteúdo em nome do Pedal Data sem autorização

## Selos e badges
Selos de parceria são fornecidos pelo Pedal Data e devem ser usados conforme:
- Tamanho mínimo legível
- Link para pedaldata.com.br
- Sem alteração de cores ou proporções
- Sem uso em produtos físicos sem autorização

## Conteúdo co-criado
- Conteúdo em parceria é identificado como "Conteúdo em parceria com [parceiro]"
- O Pedal Data mantém controle editorial
- Parceiros não aprovam o conteúdo final
- Divergências são resolvidas pela não publicação

## Revisão de materiais
Parceiros devem submeter materiais que usam a marca Pedal Data para aprovação prévia. Prazo de revisão: 5 dias úteis.
