# Arquitetura de Marca — Pedal Data

## Marca principal
**Pedal Data** — Plataforma de dados, comparação, preços e decisão.

## Submarcas

| Submarca | Função | Quando usar |
|---|---|---|
| Pedal Data Compare | Comparador de bicicletas e componentes | Quando houver comparador dedicado |
| Pedal Data Prices | Histórico de preços e alertas | Quando houver sistema de alertas e gráficos de preço |
| Pedal Data Fit | Ferramentas de tamanho, geometria e bike fit | Quando houver calculadora avançada de fit |
| Pedal Data Pro | Produto para lojas, marcas e profissionais | Quando houver painéis B2B |
| Pedal Data Lab | Pesquisas, relatórios e estudos de mercado | Quando houver relatórios pagos |

## Regra
Submarcas só devem ser usadas quando houver produto real por trás. Não criar submarcas para conceitos ou ideias.

## Relação com marca principal
Todas as submarcas são derivadas do Pedal Data. A marca principal sempre aparece em primeiro lugar.
