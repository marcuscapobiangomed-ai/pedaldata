# Modelo de Receita — Pedal Data

## Fontes de receita

### Afiliados (principal — a ativar)
- Comissão por clique ou venda em links para lojas parceiras
- Taxa varia por categoria e volume
- Margem estimada: 30-50%
- 12 lojas mapeadas como potenciais parceiras

### Leads qualificados (futuro)
- Encaminhamento de leads para lojas e oficinas
- Modelo por lead ou assinatura mensal
- Margem estimada: 50-70%

### Assinaturas premium (futuro)
- Alertas mais rápidos, histórico de preços, sem anúncios
- Mensalidade a definir
- Margem estimada: 60-80%

### Publicidade (futuro)
- Anúncios contextuais no site
- Conteúdo patrocinado identificado

### Produtos próprios (futuro)
- Relatórios de mercado (dados agregados)
- API de dados para integração
- Consultoria assistida de compra
- Margem estimada: 50-90% dependendo do produto

## Regras financeiras
- Nenhum parceiro paga para ter dados alterados ou removidos
- Conteúdo patrocinado é sempre identificado como tal
- Parceiros comerciais não têm influência editorial
- Receita é reinvestida em qualidade dos dados e conteúdo
- Nenhuma linha de receita é escalada sem margem conhecida
