# Política de Parceiros — Pedal Data

## Princípios
1. Independência editorial é inegociável
2. Parceiro pagante não recebe proteção editorial
3. Dados de parceiros passam pelo mesmo controle de qualidade
4. Parceiros são avaliados trimestralmente

## Critérios de avaliação
| Critério | Peso |
|---|---|
| Atualização de estoque | 20% |
| Precisão de preços | 25% |
| Tempo de resposta | 15% |
| Conversão | 15% |
| Reclamações | 15% |
| Transparência | 10% |

## Status de parceiro
- `active`: todos os critérios acima de 70%
- `limited`: um ou mais critérios abaixo de 50%
- `suspended`: violação de termos ou reclamações graves
- `inactive`: solicitação do parceiro

## Conflitos
- Se parceiro solicitar alteração de dados, a solicitação passa pelo mesmo processo de correção
- Decisão editorial prevalece sobre interesse comercial
- Conflitos não resolvidos são documentados publicamente

## Transparência
- Parceiros são listados publicamente
- Modelo comercial de cada parceiro é informado
- Avaliação de qualidade dos dados do parceiro é registrada
