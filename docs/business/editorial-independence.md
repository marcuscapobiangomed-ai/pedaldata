# Independência Editorial — Pedal Data

## Compromisso
O Pedal Data mantém separação total entre operação comercial e produção editorial.

## Regras
1. Nenhum parceiro, anunciante ou patrocinador revisa conteúdo antes da publicação
2. Nenhum dado é alterado por solicitação comercial sem passar pelo processo de correção
3. Conteúdo patrocinado é identificado com aviso claro e distinto do editorial
4. Produtos cedidos para teste são informados no artigo
5. Relações comerciais de autores são declaradas

## Violações
Qualquer tentativa de influenciar conteúdo editorial por meio de pressão comercial é documentada e, se relevante, tornada pública.

## Aplicação
Esta política se aplica a todo conteúdo publicado, incluindo dados, reviews, comparativos, guias e notícias.
