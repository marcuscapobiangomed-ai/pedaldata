# Produtos B2B — Pedal Data

## 1. Painel para lojas (futuro)
- Gestão de leads recebidos
- Desempenho de produtos no catálogo
- Cliques, conversão e receita por produto
- Produtos mais procurados pelos ciclistas
- Comparação de preços com o mercado
- Alerta de oportunidades (produtos sem cobertura)

## 2. Painel para marcas (futuro)
- Interesse por produto e modelo
- Comparação com concorrentes diretos
- Tendências de busca e sazonalidade
- Percepção de preço pelo consumidor
- Intenção de compra por região
- Equipamentos mais observados

## 3. API de produtos (futuro)
**Dados estruturados para integração:**
- Catálogo de bicicletas e componentes
- Preços atualizados
- Especificações técnicas
- Histórico de preços

**Potenciais clientes:**
- Lojas (integrar dados no sistema próprio)
- Marketplaces (enriquecer fichas de produto)
- Seguradoras (calcular prêmios por modelo)
- Empresas de financiamento (avaliar valor de mercado)
- Clubes e aplicativos
- Mídia especializada

## 4. Relatórios de mercado (futuro)
**Relatórios pagos:**
- Panorama do mercado brasileiro de road bikes
- Comportamento do consumidor por faixa de preço
- Distribuição de grupos e componentes
- Evolução de preços (12 meses)
- Disponibilidade regional de produtos
- Marcas mais comparadas
- Tendências de componentes

**Formato:** PDF + planilha de dados brutos
**Periodicidade:** trimestral ou anual
