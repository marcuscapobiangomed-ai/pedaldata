## Indicadores

| Indicador | Fórmula | Meta |
|---|---|---|
| MRR | Receita mensal recorrente | - |
| ARR | MRR x 12 | - |
| Margem bruta | (Receita - Custos diretos) / Receita | >70% |
| Margem operacional | (Receita - Custos totais) / Receita | >30% |
| CAC | Custo de aquisição / novos clientes | - |
| LTV | Ticket médio x meses de retenção | >3x CAC |
| Churn | Cancelamentos / total de assinantes | <5% mês |
| Payback | CAC / receita mensal média | <12 meses |
| ARPU | Receita total / total de usuários | - |
| Runway | Caixa / burn rate mensal | >12 meses |

## Cenários

### Conservador
- Crescimento orgânico lento
- Poucos parceiros
- Equipe reduzida
- Foco em eficiência

### Base
- Crescimento consistente
- Receita diversificada
- Produtos premium iniciais
- Expansão gradual

### Agressivo
- Investimento externo
- Contratações
- Expansão de catálogo
- Marketing ativo
- Novos mercados

Cada cenário deve ter receita, custos, equipe, runway, metas, riscos e gatilhos documentados separadamente.
