# Unit Economics — Pedal Data

## Afiliados

| Indicador | Fórmula |
|---|---|
| Receita por clique | Receita total / total de cliques |
| Receita por mil visitas | (Receita total / visitas) x 1000 |
| Taxa de conversão | Compras / cliques |
| Ticket médio | Receita total / compras |

## Leads

| Indicador | Fórmula |
|---|---|
| Custo por lead | Custo da campanha / leads gerados |
| Receita por lead | Receita de leads / total de leads |
| Taxa de aceitação | Leads aceitos / leads enviados |
| Taxa de venda | Vendas fechadas / leads aceitos |

## Premium

| Indicador | Fórmula |
|---|---|
| CAC | Custo de aquisição / novos assinantes |
| Mensalidade média | Receita premium / assinantes |
| Retenção média | 1 / churn rate |
| LTV | Mensalidade média / churn rate |
| Churn | Cancelamentos / total de assinantes |

## B2B

| Indicador | Fórmula |
|---|---|
| Ticket mensal médio | Receita B2B / clientes ativos |
| Custo de atendimento | Suporte + sucesso do cliente / clientes |
| Margem por cliente | (Ticket - custo atendimento) / ticket |
| Tempo de implantação | Dias da assinatura ao primeiro uso |
| Retenção | Clientes que renovaram / total de clientes |

## Regra
Nenhuma linha de receita deve ser escalada se sua margem ainda for desconhecida. Validar unit economics antes de investir em aquisição.
