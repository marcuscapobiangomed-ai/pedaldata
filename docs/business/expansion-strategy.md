# Estratégia de Expansão — Pedal Data

## Três dimensões de expansão

### Dimensão 1 — Categorias
Ordem recomendada:
1. Road bikes (atual)
2. Gravel
3. Componentes (grupos, rodas)
4. Acessórios (capacetes, sapatilhas)
5. MTB
6. Triathlon
7. E-bikes
8. Urbano

### Dimensão 2 — Geografia
Ordem recomendada:
1. Brasil (atual)
2. Capitais e regiões metropolitanas
3. Portugal (mesmo idioma)
4. América Latina (espanhol)

### Dimensão 3 — Público
Ordem recomendada:
1. Iniciantes (atual)
2. Intermediários
3. Compradores premium
4. Atletas
5. Profissionais (mecânicos, bike fitters)
6. Lojas
7. Marcas

## Regra
Nunca expandir as três dimensões ao mesmo tempo. Máximo de duas dimensões simultâneas, com uma dominante.

## Critérios para abrir nova categoria

Checklist obrigatório:
- [ ] Demanda comprovada (buscas, perguntas, tendências)
- [ ] Fontes confiáveis de dados disponíveis
- [ ] Taxonomia definida e documentada
- [ ] Especialista consultado ou disponível
- [ ] Volume mínimo de produtos (50+ para comparação)
- [ ] Oportunidade de monetização clara
- [ ] Capacidade operacional para manter
- [ ] Diferencial em relação a concorrentes
