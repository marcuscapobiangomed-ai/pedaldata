# SLA Operacional — Pedal Data

## Prazos internos

| Atividade | Prazo |
|---|---|
| Corrigir erro crítico (P0) | Imediatamente |
| Corrigir especificação importante (P1) | Até 24 horas |
| Avaliar feedback de usuário | Até 3 dias úteis |
| Atualizar preço prioritário | Até 7 dias |
| Revisar artigo vencido | Até 15 dias |
| Responder parceiro comercial | Até 1 dia útil |
| Responder usuário | Até 2 dias úteis |
| Cadastrar novo produto | Até 5 dias úteis |
| Aprovar imagem | Até 48 horas úteis |
| Onboarding de parceiro | Até 10 dias úteis |
| Correção P2 | Até 5 dias úteis |
| Correção P3 | Próxima sprint |
| Relatório mensal | Até 5º dia útil do mês |

## Monitoramento
- SLAs são registrados em relatório mensal
- Desvios > 20% no mês geram ação corretiva
- Incidentes P0 resetam contagem de SLA para o período
