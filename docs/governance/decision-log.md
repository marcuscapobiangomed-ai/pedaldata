# Registro de Decisões — Pedal Data

## Propósito
Registrar decisões estratégicas, seu contexto e impacto.

## Formato

| Data | Decisão | Contexto | Responsável | Status |
|---|---|---|---|---|
| - | - | - | - | - |

## Decisões estratégicas (exigem registro formal)
- Entrada em nova categoria de produto
- Contratação relevante
- Mudança de modelo comercial
- Captação de investimento
- Venda de participação
- Parceria exclusiva
- Mudança de marca
- Aquisição de empresa

## Decisões operacionais (não exigem registro)
- Publicação de conteúdo
- Atualização de preços
- Correções
- Ajustes de plataforma
- Suporte a usuários
- Parcerias de afiliado padrão
