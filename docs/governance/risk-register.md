# Registro de Riscos — Pedal Data

## Matriz de riscos

| Risco | Probabilidade | Impacto | Mitigação | Responsável |
|---|---|---|---|---|
| Conteúdo incorreto | Média | Alto | Revisão, fontes verificadas, checklist | Editorial |
| Dependência de SEO | Alta | Alto | Diversificar canais (newsletter, redes, B2B) | Produto |
| Dependência de afiliados | Média | Alto | Desenvolver leads, B2B e produtos próprios | Comercial |
| Concorrente copiar | Alta | Médio | Dados proprietários, rede de parceiros, comunidade | Produto |
| Perda de parceiro grande | Média | Médio | Diversificação de parceiros | Comercial |
| Falha de segurança | Baixa | Crítico | Controles de acesso, LGPD, auditoria | Tecnologia |
| Baixa retenção de usuários | Média | Alto | Produto recorrente (alertas, favoritos, premium) | Produto |
| Custos de IA | Média | Médio | Limites de uso, otimização, modelo próprio | Tecnologia |
| Dependência de fontes de dados | Média | Alto | Múltiplas fontes, dados proprietários, histórico | Dados |
| Mudança de algoritmo de busca | Média | Médio | SEO diversificado, tráfego direto, marca | Marketing |
| Problemas LGPD | Baixa | Alto | Políticas, consentimento, DPO | Jurídico |
| Erro de preço em larga escala | Baixa | Alto | Validação automatizada, sinalização de outlier | Dados |

## Monitoramento
Riscos revisados trimestralmente. Gatilhos e planos de ação documentados em board notes.
